# array\_prepend

## Funzione `array_prepend`

Restituisce un array con il valore dato aggiunto all'inizio.

## Sintassi

* array_prepend\(\_array, value_\)

## Argomenti

* _array_ un array
* _value_ il valore da aggiungere

## Esempi

* `array_prepend(array(1,2,3),0) â†’ array: 0,1,2,3`
