# array\_get

## Funzione `array_get`

Restituisce il valore ennesimo \(0 per il primo elemento\) di un array.

## Sintassi

* array_get\(\_array, index_\)

## Argomenti

* _array_ un array
* _index_ l'indice da estrarre \(partendo da 0\)

## Esempi

* `array_get(array('a','b','c'),1) â†’ 'b'`
