# uuid

## funzione `uuid`

Genera un Identificatore Universale Unico \(UUID\) per ogni riga usando il metodo Qt QUuid::createUuid. Ciascun UUID è lungo 38 caratteri.

## Sintassi

* uuid\(\)

## Esempi

```text
* uuid() → '{0bd2f60f-f157-4a6d-96af-d4ba4cb366a1}'
```
